# Naming Conventions

General:
- Use `snake_case` for Python variables and filenames
- Use `PascalCase` for Python classes and React components
- Use `camelCase` for JavaScript/TypeScript variables and function names

Backend Examples:
- Model: `Project`, `Estimate`
- Schema: `ProjectCreate`, `EstimateResponse`

Frontend Examples:
- Component: `ProjectCard.tsx`, `EstimateForm.tsx`
- Hook: `useProjectForm()`
